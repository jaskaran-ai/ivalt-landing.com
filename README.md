# ivalt-landing.com

This is the landing page for ivalt.com. It is built with Next.js, Typescript, and Tailwind CSS.

## Getting Started

First, install the dependencies:

```bash
npm install
# or
yarn install
```

To run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Building

To build the project:

```bash
npm run build
# or
yarn build
```

The built files will be in the `out` directory.

## Deployment

This project is deployed on Vercel. The deployment configuration is in the `vercel.json` file.

To deploy manually, you can use the Vercel CLI:

```bash
vercel --prod
```

To deploy automatically, every time you push to the `main` branch, you can use the following command:

```bash
git push origin main
```
